# Book Management System in C++

A simple console-based book management system to add, search, remove, and save books.

## Features

- Add new books (title & author)
- Search by book title or author name
- Remove books by index
- Show all books in a formatted table
- Save books to a CSV file
- Load books from a CSV file

## Menu

1) Add
2) Search
3) Remove
4) Show
5) Save
6) Load
7) Exit

Press any key to continue after each operation

## File Format (CSV)

When saving, books are stored as:

```
Book title,Book Author
Harry Potter,J.K. Rowling
1984,George Orwell
```

## Limitations

- Windows only
- Case-sensitive search
- No duplicate checking
